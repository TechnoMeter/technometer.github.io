# Windows Vista Aero Portfolio

A nostalgic, high-fidelity recreation of the Windows Vista "Aero" interface, built as a personal portfolio. This project leverages modern web technologies to simulate a desktop environment, complete with translucent glass effects, draggable windows, and functional mini-apps.

## 🌟 Features

- **Aero Glass UI**: Authentically recreated glassmorphism using Tailwind CSS and backdrop filters, mimicking the classic Windows Vista aesthetic.
- **Windows Sidebar (Gadget Gallery)**: 
  - Recreated the iconic translucent side panel with hover-triggered deep glass effects.
  - **Analog Clock**: Real-time ticking clock with authentic Vista dial styling and light reflections.
  - **Calendar**: Signature red-header calendar dynamically highlighting the current date.
  - **Resource Meter**: Simulated CPU and RAM usage gauge with animated SVGs and drop-shadow needles.
- **Window Management System**: 
  - Draggable, resizable, and stackable windows with viewport boundary constraints (windows won't get lost off-screen).
  - Functional Minimize, Maximize, and Close controls.
  - Focus management (clicking a window brings it to the front with active drop shadows).
- **Interactive Desktop**:
  - Double-click (or single-click on mobile) desktop icons to launch apps or external links (GitHub, LinkedIn, YouTube).
- **Functional Mini-Apps**:
  - **My Computer (About Me)**: A customized profile card featuring system information, a profile picture, and CV download links.
  - **Paint**: A fully functional canvas-based drawing application with color selection and brush controls.
  - **Calculator**: A working arithmetic calculator with basic operations.
  - **Notepad (README)**: A classic text-editor view detailing project inspiration and personal hobbies.
  - **File Explorer**: A projects showcase organized in a folder-style layout with tech stack tags.
  - **Control Panel**: A categorized view of educational background and technical skills.
- **Dynamic Taskbar & Start Menu**:
  - Live clock and system tray indicators.
  - Functional Start Orb with user profile, quick links, and a searchable-style menu.
  - Active task tracking in the taskbar with minimize/restore functionality.
- **Responsive Design**: Adapts the desktop experience seamlessly for mobile and tablet views (e.g., auto-hiding the Sidebar on smaller screens).

## 🚀 Technologies Used

- **React**: Component-based architecture for the OS state and window management.
- **Tailwind CSS**: Custom utility classes for the "Aero" glass effects, gradients, and layout.
- **Lucide React**: Comprehensive icon set used for desktop and system icons.
- **Vite**: Ultra-fast frontend tooling for development and bundling.
