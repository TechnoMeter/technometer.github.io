# Windows Vista Aero Portfolio

A nostalgic, high-fidelity recreation of the Windows Vista "Aero" interface, built as a personal portfolio. This project leverages modern web technologies to simulate a desktop environment, complete with translucent glass effects, draggable windows, and functional mini-apps.

## 🌟 Features

- **Aero Glass UI**: Authentically recreated glassmorphism using Tailwind CSS and backdrop filters, mimicking the classic Windows Vista aesthetic.
- **Window Management System**: 
  - Draggable, resizable, and stackable windows.
  - Functional Minimize, Maximize, and Close controls.
  - Focus management (clicking a window brings it to the front).
- **Functional Mini-Apps**:
  - **Paint**: A fully functional canvas-based drawing application with color selection.
  - **Calculator**: A working arithmetic calculator with basic operations.
  - **Notepad (README)**: A classic text-editor view for introductory text.
  - **File Explorer**: A projects showcase organized in a folder-style layout.
  - **Control Panel**: A categorized view of education and technical skills.
- **Dynamic Taskbar & Start Menu**:
  - Live clock and system tray.
  - Functional Start Orb with a searchable-style menu.
  - Active task tracking in the taskbar.
- **Responsive Design**: Adapts the desktop experience for mobile and tablet views.

## 🚀 Technologies Used

- **React**: Component-based architecture for the OS state and window management.
- **Tailwind CSS**: Custom utility classes for the "Aero" glass effects, gradients, and layout.
- **Lucide React**: Comprehensive icon set used for desktop and system icons.
- **Vite**: Ultra-fast frontend tooling for development and bundling.
